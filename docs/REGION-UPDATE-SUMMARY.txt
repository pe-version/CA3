│                      HOW TO FIND YOUR AD STRING                            │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                            │
│  Method 1: OCI Console → Governance → Limits                              │
│    1. Click "Compute"                                                     │
│    2. Find "VM.Standard.A1.Flex - OCPU Count"                             │
│    3. Look for AD with Available > 0                                      │
│    4. Note the name (e.g., "US-CHICAGO-1-AD-1")                           │
│    5. Full format: YXRh:US-CHICAGO-1-AD-1                                 │
│                                                                            │
│  Method 2: Start Instance Creation                                        │
│    1. OCI Console → Compute → Instances → Create                          │
│    2. Under "Placement", dropdown shows exact AD strings                  │
│    3. Copy entire string (includes YXRh: prefix)                          │
